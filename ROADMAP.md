# 🚀 Iconary Roadmap (2026)

Welcome to the Iconary Roadmap! We have an ambitious vision to become the standard open-source icon library for the AI and ML ecosystem.

Here is what we are currently working on and what we plan to build next. If you are looking to contribute, anything on this list is a great place to start!

## 📌 Phase 1: Core Library Expansion (Current)
- [ ] **Expand Icon Coverage**: Reach 500+ AI/ML specific icons.
- [ ] **Data Science Concepts**: Add specific icons for dataframes, tensors, classification, regression, etc.
- [ ] **Framework Integration**: Ensure robust support for React, Vue, and Svelte components.

## 🛠️ Phase 2: Tooling & Automation
- [x] Set up CI/CD for automated linting and building (Completed)
- [ ] **Figma Plugin**: Build a Figma plugin so designers can drag and drop Iconary icons directly into their designs.
- [ ] **SVG Optimizer CLI**: A script to automatically run all new SVGs through SVGO to ensure maximum performance before they are merged.
- [ ] **Website Redesign**: A dedicated documentation site with a searchable icon gallery and live playground.

## 🌟 Phase 3: Community & Education
- [ ] **Concept Explainers**: Complete the "1-2 line plain-English explanations" for every single icon.
- [ ] **Interactive Tutorials**: Visual guides explaining complex AI architectures using our icons.

---

### How to Contribute
Want to tackle something on this roadmap? 
1. Check the [Issues tab](https://github.com/pradhan-not-found/Iconary/issues) to see if someone is already working on it.
2. If not, open an issue proposing your implementation.
3. Check out `CONTRIBUTING.md` for guidelines!
