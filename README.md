# Iconary

> Open source icons for AI, ML & data tooling — with built-in concept explanations.

Iconary is a growing collection of icons designed specifically for AI/ML products, dashboards, and developer tools. Every icon comes in **outline, filled, and duotone** styles, and includes a short explanation of the concept it represents — making it equally useful for designers, developers, and anyone learning AI/ML terminology.

## Why Iconary?

- 🎯 **Built for AI/ML** — icons for concepts like agents, embeddings, fine-tuning, RAG, vector databases, and more.
- 🎨 **3 styles from 1 source** — outline, filled, and duotone variants generated automatically.
- 📖 **Concept explainers** — each icon includes a 1-2 line plain-English explanation.
- 📦 **Framework-ready** — React, Vue, and Svelte packages (Figma plugin coming soon).
- 🌱 **Always growing** — new AI/ML terms get new icons; request one or contribute your own.

## Quick Start

```bash
npm install @iconary/react
```

```jsx
import { Agent, VectorDatabase, FineTuning } from "@iconary/react";

function App() {
  return <Agent style="duotone" size={24} />;
}
```
