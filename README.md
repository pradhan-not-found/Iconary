## 🌟 About Iconary
Iconary is a meticulously crafted, open-source icon set designed explicitly for the AI era. With an ever-growing ecosystem of agents, vector databases, pipelines, and models, Iconary provides developers and designers with the visual primitives they need to build beautiful tooling.

- ⚡️ **Zero Friction:** Available as raw SVGs, or ready-to-use React, Vue, and Svelte components.
- 🎨 **Beautifully Designed:** Consistent stroke widths, sleek radii, and a premium aesthetic perfect for dark-mode data dashboards.
- 🤝 **Community Driven:** Anyone can request an icon. Tiny contributions (1 icon = 1 PR) are highly encouraged!

## 🚀 Quick Start (React)

## 🛠 Project Structure
Our open source monorepo uses Vite for a lightning-fast showcase gallery:
- `icons/` - The raw SVG source files and `registry.json` database.
- `packages/` - Framework specific wrappers (React, Vue, Svelte).
- `src/` - Our gorgeous dark-mode showcase gallery built on React + Tailwind CSS v4 + Vite.

## 🤝 Contributing
We love contributions! The easiest way to get started is to grab an issue tagged with `good first issue` or submit a new SVG icon for a missing AI/Data concept.

1. Pick an open icon request.
2. Draw it using our style guide (24x24 viewBox, 1.5px stroke).
3. Submit a PR!
